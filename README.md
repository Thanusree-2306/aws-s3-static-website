# aws-s3-static-website
This project demonstrates how to host a static website using Amazon S3 in the AWS Management Console.
It includes two basic HTML files — index.html (the homepage) and error.html (the custom error page).
The website is configured for static hosting with public access and tested successfully using the S3 bucket website endpoint.

Key Features

Hosted on Amazon S3 (Simple Storage Service)

Includes custom index and error pages

Uses S3 static website hosting

Demonstrates AWS cloud storage and hosting concepts

Files

index.html – Main homepage

error.html – Displayed for invalid URLs or missing pages

Learning Outcome

Understand how to create and configure an S3 bucket

Learn to upload website files and enable static website hosting

Gain hands-on experience with cloud-based web hosting on AWS
